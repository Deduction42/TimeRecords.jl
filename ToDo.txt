4.  aggregate(ts::TimeSeries, t::AbstractVector{<:Real}) now returns only interpolated values, not TimeSeries
5.  integrate(ts::TimeSeries, t::AbstractVector{<:Real}) now returns only interpolated values, not TimeSeries
